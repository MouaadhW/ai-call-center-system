# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-06

### Added
- Initial release
- Asterisk PBX integration for VoIP call handling
- Whisper ASR for speech-to-text
- Piper TTS for text-to-speech
- AI agent with intent classification
- Knowledge base system
- SQLite database with customer, call, and ticket models
- FastAPI REST API
- React dashboard with analytics
- Docker containerization
- Complete documentation (README, INSTALLATION, QUICKSTART, TESTING)
- Sample data seeding
- Test scripts

### Features
- Automatic call answering
- Customer verification
- Intent detection (billing, technical support, account info)
- Support ticket creation
- Real-time analytics dashboard
- Call transcripts
- Multi-user SIP support

### Documentation
- Installation guide
- Quick start guide
- Testing procedures
- API documentation
- Contributing guidelines
